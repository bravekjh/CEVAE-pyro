# CEVAE-pyro
Pytorch and Pyro implementation of CEVAE(Causal Effect Variational AutoEncoder). See https://arxiv.org/pdf/1705.08821.pdf
# Run
```bash
python3 main.py
```
# Environment
Python 3.6
Numpy 1.16.4
Pytorch 1.1.0
Pyro 0.3.3
